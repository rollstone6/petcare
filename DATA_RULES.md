## 2. 商品标签规则

### 2.1 物种 (suitable_species)

| 值 | 含义 | 数量 |
|----|------|------|
| 猫狗 | 猫狗通用 | 113 |
| 猫 | 仅猫 | 39 |
| 狗 | 仅狗 | 34 |

**规则：** 值只能是 "猫"/"狗"/"猫狗"，不允许"犬"、"猫,犬"等变体。

### 2.2 体型 (target_size)

| 值 | 含义 | 适用场景 |
|----|------|---------|
| 全部 | 不区分体型 | 药品、保健品、通用食品 |
| 小型 | 小型犬 (<10kg) | 小型犬专用粮/用品 |
| 中型 | 中型犬 (10-25kg) | 很少单独使用 |
| 大型 | 大型犬 (>25kg) | 大型犬关节药等 |
| 小型中型 | 中小型犬 | 幼犬粮（长大变中型） |
| 中大型 | 中大型犬 | 大型犬幼犬粮 |

**自动推断规则（按商品名关键词）：**
- 含"大型犬" → 大型
- 含"小型犬" → 小型
- 含"幼犬" → 小型中型
- 含"中大型" → 中大型

### 2.3 年龄 (target_age)

| 值 | 含义 | 适用场景 |
|----|------|---------|
| 全部 | 全年龄段 | 大部分药品/保健品 |
| 幼年 | 幼猫/幼犬 | 幼犬粮、幼猫粮、奶糕 |
| 成年 | 成猫/成犬 | 成猫粮、成犬粮 |
| 老年 | 老年宠物 | 老年猫粮、关节保健品 |
| 成年老年 | 成年+老年 | 部分处方粮 |

**自动推断规则（按商品名关键词）：**
- 含"幼猫"/"幼犬"/"奶糕" → 幼年
- 含"老年" → 老年
- 含"成猫"/"成犬" → 成年

## 4. 品种关联规则 (breed_product)

### 4.1 关联逻辑

```
商品适合物种 → 候选品种池
  ├─ "猫狗" → 全部62个品种
  ├─ "猫"   → 28个猫品种
  └─ "狗"   → 34个狗品种

体型过滤（仅对狗生效）：
  ├─ target_size="小型"     → 仅11个小型犬
  ├─ target_size="小型中型" → 22个中小型犬
  ├─ target_size="大型"     → 仅12个大型犬
  ├─ target_size="中大型"   → 23个中大型犬
  └─ target_size="全部"     → 全部34个狗品种

猫品种不做体型过滤（统一中型）。
```

### 4.2 关联示例

| 商品 | 物种 | 体型 | 关联品种数 |
|------|------|------|-----------|
| 皇家小型犬情绪舒缓 | 狗 | 小型 | 11个小型犬 |
| 皇家GIJ29幼犬处方粮 | 狗 | 小型中型 | 22个中小型犬 |
| 皇家LP18成犬泌尿道 | 狗 | 全部 | 34个全部狗 |
| 皇家SC36老年猫粮 | 猫 | 全部 | 28个全部猫 |
| 大宠爱（驱虫） | 猫狗 | 全部 | 62个全部品种 |

### 4.3 通用商品不单独筛选

药品、保健品等"全物种/全体型/全年齢"商品关联所有匹配品种。
前端展示时通过分类+搜索来筛选，而非限制关联数。

## 6. 数据导入流程

### 6.1 京东数据抓取

```
API: https://api.m.jd.com/client.action
functionId: whx_searchWare
shopId: 16883392
pageSize: 30
```

需要有效的cookie（thor、pin、__jda等），cookie有效期约30分钟。

### 6.2 导入步骤

1. **抓取JSON** → curl请求API，保存为 /tmp/jd_pageN.json
2. **去重检查** → 按 wareId 去重，跳过已存在商品
3. **创建品牌** → 新品牌自动创建，国家手动标注
4. **分类映射** → 按商品内容手动分类（京东分类名不准）
5. **标签推断** → target_size/target_age 按关键词自动推断
6. **品种关联** → 按物种+体型规则自动关联
7. **图片补充** → 从JSON提取 imageurl

### 6.3 京东商品ID标记

- 存储在 `Product.approval_number` 字段，格式 `JD:{wareId}`
- 用于去重和数据溯源
