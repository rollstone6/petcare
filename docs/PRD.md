## 1. 产品定位

宠物宝是一个**跨端（Web + 移动端 PWA + 未来小程序/App）**宠物产品成分查询与安全评分平台。帮助宠物主在购买前了解产品成分、安全性和适用性。

### 对标美丽修行的核心能力

| 美丽修行功能 | 宠物宝对应功能 |
|-------------|---------------|
| 化妆品成分查询 | 宠物药品/食品/保健品成分查询 |
| 安全评分（0-5分） | 安全评分（0-5分） |
| 肤质测试 → 产品推荐 | 品种档案 → 产品推荐 |
| 成分百科 | 成分百科 |
| 产品对比 | 产品对比 |
| 用户评价/心得 | 用户评价/心得 |
| 收藏夹 | 收藏夹 |
| 扫码查产品 | 扫码查产品（后续） |
| 电商导购 | 电商导购（后续） |

## 3. 安全评分算法（对标美丽修行）

美丽修行的评分基于成分安全等级加权计算。宠物宝的评分算法：

```
产品安全评分 = Σ(成分安全等级 × 成分权重) / 成分总数 × 品类系数

成分安全等级：
  5 = 🟢 非常安全（疫苗、必需营养素）
  4 = 🟢 安全（常见驱虫成分、营养补充）
  3 = 🟡 注意（抗生素、激素类）
  2 = 🟠 谨慎（有已知副作用）
  1 = 🔴 风险（禁用/慎用成分）

品类系数：
  药品: 0.9（药品本身有风险，降低期望）
  食品: 1.0
  保健品: 1.0

评分维度：
  - 成分安全分 (40%)
  - 适用性分 (30%) — 是否适合目标品种/年龄
  - 用户口碑分 (20%)
  - 品牌信誉分 (10%)
```
