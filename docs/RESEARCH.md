### 实际采用的数据源

**京东店铺：浙大动物医院专营店 (shopId=16883392)**
- 5页共138个商品（药品+保健品+处方粮+普通食品）
- 包含52个品牌（皇家23、PEMO颢宸18、信得药业PE14等）
- 含商品图片、价格、分类信息
- 详见 DATA_RULES.md 第6节"数据导入流程"

### 未采用的开源资源（保留参考）

- **Open Pet Food Facts** — 900+产品，但主要是海外品牌，缺乏中国宠物市场数据
- **Dog API / Cat API** — 品种数据英文为主，缺少中文品种名和常见健康问题
- **中国兽药信息网** — 需爬虫，且不公开完整成分表

## 三、分阶段实施计划

### Phase A — 数据导入（本周）
```
优先级：最高 | 工作量：2-3天
```

**A1. 从 Open Pet Food Facts 导入宠物食品**
- [ ] 编写导入脚本，批量拉取 900+ 产品
- [ ] 成分名中英文映射表
- [ ] 品牌信息自动创建
- [ ] 品类自动分类（猫粮/狗粮/零食/处方粮）
- [ ] 目标：产品库从 29 → 500+

**A2. 从 Dog API 导入品种数据**
- [ ] 导入 100+ 犬种基础数据
- [ ] 品种名中英文翻译
- [ ] 补充常见健康问题（整合兽医文献）
- [ ] 目标：品种从 24 → 120+

**A3. 从 Cat API 导入猫品种**
- [ ] 导入 60+ 猫品种
- [ ] 品种名翻译
- [ ] 补充健康问题
- [ ] 目标：猫品种从 10 → 60+

---

### Phase B — 算法优化（下周）
```
优先级：高 | 工作量：2-3天
```

**B1. 参考 Open Food Facts 评分体系**
- [ ] 研究 Nova Score 算法（食品加工程度评分）
- [ ] 研究 Eco-Score 算法（环境影响评分）
- [ ] 适配为宠物安全评分：
  - 成分安全分（基于成分安全等级）
  - 营养均衡分（基于营养成分分析）
  - 添加剂风险分（基于添加剂标签）
  - 品牌信誉分

**B2. 引入成分分析标签**
- [ ] 参考 Open Beauty Facts 的 ingredients_analysis_tags
- [ ] 建立宠物专用标签体系：
  - 含谷物 / 无谷
  - 含人工添加剂 / 天然
  - 高蛋白 / 低脂
  - 适合幼年 / 成年 / 老年
  - 处方级 / 日常级

---

### Phase C — 前端增强（第三周）
```
优先级：中 | 工作量：2-3天
```

**C1. 产品对比增强**
- [ ] 营养成分并排对比
