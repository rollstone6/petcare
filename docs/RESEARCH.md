# 宠物宝 — 开源资源调研与内容规划

> 版本：v1.0 | 最后更新：2026-06-28

## 一、已验证的开源资源

### 1. Open Pet Food Facts ⭐⭐⭐⭐⭐
| 项目 | 详情 |
|------|------|
| **API** | `https://world.openpetfoodfacts.org/api/v2/` |
| **数据规模** | 936+ 宠物食品产品（持续增长） |
| **数据字段** | 产品名、品牌、完整成分表、营养成分、添加剂、过敏原 |
| **许可** | ODbL（开放数据） |
| **可复用度** | ⭐⭐⭐⭐⭐ 直接可用 |

**可直接导入的内容：**
- 900+ 款猫粮/狗粮产品及完整成分表
- 品牌信息（Acana、Royal Canin 等国际品牌）
- 营养成分分析（蛋白质、脂肪、纤维等）
- 添加剂和过敏原标签

**需要适配的工作：**
- 英文产品名 → 中文翻译
- 成分名标准化（英文 → 中文 INCI 名）
- 添加安全评分
- 关联品种推荐

---

### 2. Dog API ⭐⭐⭐⭐
| 项目 | 详情 |
|------|------|
| **API** | `https://dogapi.dog/api/v2/breeds` |
| **数据规模** | 100+ 犬种 |
| **数据字段** | 品种名、描述、寿命范围、体重范围、是否低敏 |
| **许可** | 免费公开 |
| **可复用度** | ⭐⭐⭐⭐ |

**可直接导入的内容：**
- 100+ 犬种的标准化数据
- 品种描述（英文）
- 寿命和体重范围
- 低敏标识

**需要适配的工作：**
- 品种名翻译
- 补充常见健康问题（需从兽医文献整合）
- 关联推荐产品

---

### 3. Open Beauty Facts ⭐⭐⭐⭐⭐
| 项目 | 详情 |
|------|------|
| **API** | `https://world.openbeautyfacts.org/api/v2/` |
| **数据规模** | 65,000+ 化妆品 |
| **数据字段** | 产品名、品牌、完整成分表、成分分析标签 |
| **许可** | ODbL |
| **可复用度** | ⭐⭐⭐⭐⭐ 架构参考 |

**参考价值（不直接导入数据，因为是化妆品）：**
- 成分分析标签系统（ingredients_analysis_tags）
- 产品搜索和筛选架构
- API 设计模式
- 开源评分算法（Nova Score、Eco-Score）

---

### 4. 中国兽药信息网 (ivdc.org.cn)
| 项目 | 详情 |
|------|------|
| **数据规模** | 数万条兽药批准文号 |
| **可复用度** | ⭐⭐⭐ 需爬取 |

**可直接获取：**
- 兽药产品名称
- 生产企业
- 批准文号
- 规格

**限制：**
- 不公开完整成分表
- 需要爬虫采集

## 三、分阶段实施计划

### Phase A — 数据导入（本周）
```
优先级：最高 | 工作量：2-3天
```

**A1. 从 Open Pet Food Facts 导入宠物食品**
- [ ] 编写导入脚本，批量拉取 900+ 产品
- [ ] 成分名中英文映射表
- [ ] 品牌信息自动创建
- [ ] 品类自动分类（猫粮/狗粮/零食/处方粮）
- [ ] 目标：产品库从 29 → 500+

**A2. 从 Dog API 导入品种数据**
- [ ] 导入 100+ 犬种基础数据
- [ ] 品种名中英文翻译
- [ ] 补充常见健康问题（整合兽医文献）
- [ ] 目标：品种从 24 → 120+

**A3. 从 Cat API 导入猫品种**
- [ ] 导入 60+ 猫品种
- [ ] 品种名翻译
- [ ] 补充健康问题
- [ ] 目标：猫品种从 10 → 60+

---

### Phase B — 算法优化（下周）
```
优先级：高 | 工作量：2-3天
```

**B1. 参考 Open Food Facts 评分体系**
- [ ] 研究 Nova Score 算法（食品加工程度评分）
- [ ] 研究 Eco-Score 算法（环境影响评分）
- [ ] 适配为宠物安全评分：
  - 成分安全分（基于成分安全等级）
  - 营养均衡分（基于营养成分分析）
  - 添加剂风险分（基于添加剂标签）
  - 品牌信誉分

**B2. 引入成分分析标签**
- [ ] 参考 Open Beauty Facts 的 ingredients_analysis_tags
- [ ] 建立宠物专用标签体系：
  - 含谷物 / 无谷
  - 含人工添加剂 / 天然
  - 高蛋白 / 低脂
  - 适合幼年 / 成年 / 老年
  - 处方级 / 日常级

---

### Phase C — 前端增强（第三周）
```
优先级：中 | 工作量：2-3天
```

**C1. 产品对比增强**
- [ ] 营养成分并排对比
- [ ] 成分差异高亮
- [ ] 评分雷达图

**C2. 搜索增强**
- [ ] 全文搜索（FTS5）
- [ ] 按成分筛选（含/不含某成分）
- [ ] 按营养指标筛选（高蛋白、低脂等）
- [ ] 按适用阶段筛选（幼年/成年/老年）

**C3. 品种页面增强**
- [ ] 品种详情页展示寿命、体重范围
- [ ] 低敏品种标识
- [ ] 品种对比功能
