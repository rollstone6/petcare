## 5. 交互设计

### 5.1 搜索
- 输入即搜索（debounce 300ms）
- 搜索结果下拉预览（前5条）
- 回车跳转完整搜索结果页

### 5.2 加载状态
- 骨架屏（Skeleton）替代 spinner
- 图片懒加载

### 5.3 空状态
- "暂无数据" + 插画
- 搜索无结果："换个关键词试试"

### 5.4 错误状态
- Toast 提示网络错误
- 重试按钮
