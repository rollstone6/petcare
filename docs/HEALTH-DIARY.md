# 宠物宝 PetCare — 宠物日常健康日记

> 版本：v1.0 | 对标：宠物健康追踪（便便/饮水/呕吐/体重/食欲等）

---

## 一、功能概述

宠物主可以记录宠物的日常健康状况，通过时间线查看历史记录。

### 支持的记录类型
| 类型 | 专属字段 |
|------|---------|
| 💩 便便 | 颜色（棕/黑/红/绿/黄/白）+ 形状（正常/软便/稀便/硬便/带血） |
| 💧 饮水 | 饮水量（ml） |
| 🤮 呕吐 | 呕吐物（食物/黄水/白沫/血丝/异物）+ 严重程度 |
| ⚖️ 体重 | 体重数值（kg） |
| 🍖 食欲 | 严重程度（正常/注意/严重） |
| 😊 精神 | 严重程度 |
| 💊 用药 | 药品名称+剂量 |

### 严重程度
- 🟢 normal — 正常
- 🟡 warning — 注意
- 🔴 danger — 严重

## 三、API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/records | 创建记录 |
| GET | /api/records | 记录列表（支持类型筛选、分页） |
| GET | /api/records/stats | 统计（by_type, severity） |
| DELETE | /api/records/{id} | 删除记录 |

---

## 四、数据模型

```sql
health_records (
  id, user_id, pet_name,
  record_type,      -- 便便/饮水/呕吐/体重/食欲/精神/用药/其他
  value,            -- 数值
  note,             -- 备注
  poop_color,       -- 便便颜色
  poop_shape,       -- 便便形状
  vomit_type,       -- 呕吐物类型
  severity,         -- normal/warning/danger
  recorded_at,      -- 记录时间
  created_at
)
```
